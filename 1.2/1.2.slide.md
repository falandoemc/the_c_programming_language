# Como a linguagem C reage um número inteiro dividido por um número inteiro?

### Como:
5/9 = 0.5555555....
### Logo:
5/9 = 0
#### Assim todas as temperaturas dariam 0 por isso não deixamos dessa forma no nosso codigo.
